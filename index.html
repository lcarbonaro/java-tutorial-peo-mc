Java was originally called Oak, after a tall oak tree outside James Gosling's office window.

It was soon dicovered that another company, called Oak Technologies, already had rights to that name, and so James and his team at Sun had to come up with another name.

double random;
int limit;

// generate a random number
random = Math.random();
System.out.println( random ); // between 0 and 1 (but never 1)

// random between 0 and n-1
limit = 100;
random = Math.random() * limit;
System.out.println( random ); // between 0 and 99

// but we want integers
random = Math.random() * limit;
System.out.println( (int) random ); // integer between 0 and 99

// but we want integers between 1 and 100
random = ( Math.random() * limit ) + 1;
System.out.println( (int) random ); // integer between 1 and 100

Arrays

A list or group of values

Multiple values in one variable

- fixed number of values

- same data type

- access by zero-based index

- can be multi-dimensional

An array is a fixed-size collection of elements of the same type.
A way to store multiple values in one variable.

// as separate variables
String name1 = "Tom";
String name2 = "Richard";
String name3 = "Harriet";
// as an array
String[] names = new String[]{"Tom","Richard","Harriet"};

to declare with known values, see names example above
to declare empty array
int[] numbers = new int[3];
numbers[0] = 100;
numbers[1] = 401;
numbers[2] = 42;
numbers[3] = 13; // will give out-of-bounds error

Fixed size is determined at the time of declaration.
Once an array is created, its size cannot be changed.
Can check size using .length property (not method)
Access via (zero-based) index System.out.println( names[2] );
Can change values via index too
names[0] = "Thomas";
